# Color Standards

> **Version:** 1.0  
> **Status:** Living Standard  
> **Last Updated:** July 2026

## Principles

- Use semantic tokens rather than raw colors.
- Maintain contrast in light and dark mode.
- Never communicate status through color alone.
- Reserve strong colors for meaning and action.

## Semantic Roles

- Background
- Surface
- Border
- Primary text
- Secondary text
- Primary action
- Success
- Warning
- Destructive
- Informational
- Muted

## Status Usage

- Success: approved, active, complete
- Warning: pending, expiring, attention required
- Destructive: rejected, failed, overdue
- Informational: scheduled, draft, neutral workflow state

## Charts

Use distinguishable series, labels, legends, and accessible tooltip text. Avoid relying on similar shades alone.
