# UI Design System

> **Version:** 1.0  
> **Status:** Living Standard  
> **Last Updated:** July 2026

## Principles

- Clear before decorative
- Consistent across modules
- Accessible by default
- Responsive from the start
- Dense enough for HR operations
- Calm, professional visual language

## Foundations

- Tailwind CSS
- Shared UI components
- Lucide icons
- Light and dark themes
- Semantic design tokens

## Layout

- Persistent dashboard navigation
- Clear page title and breadcrumbs
- Primary action near the title
- Summary cards for key metrics
- Filters above operational tables
- Consistent empty, loading, and error states

## Interaction

- Confirm destructive actions
- Show immediate success or error feedback
- Preserve filter state in URLs where useful
- Use dialogs for focused tasks
- Use full pages for complex workflows

## Accessibility

Meet WCAG-aligned standards for contrast, keyboard navigation, labels, focus, semantics, and status communication.
