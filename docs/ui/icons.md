# Icon Standards

> **Version:** 1.0  
> **Status:** Standard  
> **Last Updated:** July 2026

## Library

AI HR Vault uses `lucide-react` as the primary icon library.

## Rules

- Use icons to reinforce labels, not replace them in ambiguous actions.
- Provide accessible names for icon-only buttons.
- Keep size consistent within the same context.
- Avoid mixing icon libraries without a documented reason.
- Use semantic icons consistently across modules.
