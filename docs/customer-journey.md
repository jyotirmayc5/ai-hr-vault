# Journey Principles

Every new feature should answer these questions:

1. Which stage of the customer journey does this improve?
2. Does it reduce effort for the customer?
3. Does it increase confidence and trust?
4. Can a first-time user understand it without training?
5. Does it help customers achieve value faster?

If the answer to these questions is "no," reconsider the feature before building it.
